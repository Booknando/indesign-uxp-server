# InDesign UXP Bridge — Roadmap

## Project Goal
Build a UXP plugin + Node.js WebSocket bridge that enables external tools (MCP servers, scripts) to control InDesign via the UXP API.

## Milestones

### Milestone 1: UXP Bridge Phase 1 — Plugin + Bridge Scaffold
**Goal:** Minimal working UXP plugin and WebSocket bridge server. Prove the round-trip: external Node.js server ↔ UXP plugin ↔ InDesign DOM.
